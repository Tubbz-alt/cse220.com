Terminology & Nomenclature

  • connective — An operator in a logical expression joining terms and literals, such as AND, OR, NOT.
  • literal — A single input to a logical expression.
  • term —A grouping of literals.
  • minterm — A term whose output is the minimum value of its inputs (literals). Also, a term that appears in a Sum-of-Products expression.
  • maxterm — A term whose output is the maximum value of its inputs (literals). Also, a term that appears in a Product-of-Sums expression.

Boolean Logic as a Many-Valued Logic

Boolean logic is concerned with two values: true and false (or alternatively, one and zero). Boolean logic is therefore a binary logic. There are other logics that use more values; these are called many-valued logics. For example, Kleene logic uses three values: true, unknown, and false. Other logics may use more, up to infinitely many values to represent degrees of truth.

These truth values form a set: {T, F} for Boolean logic, and {T, U, F} for Kleene logic. With many truth values comes the question of how to define the behavior of logical connectives. Kleene logic and Boolean logic share a generalized definition for their AND (conjunction) and OR (disjunction) connectives. They define truth values to be comparable as follows:

Logic Truth Value Ordering
Boolean T > F
Kleene T > U > F
